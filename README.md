ACS: AutoCatSitter
===
Watch your cat/other pet while you're away. 

Current features
---
- Capture & save picture upon motion detection
- Livestream with web interface (WIP)

Hardware requirements
---
- Raspberry Pi Zero/3B+
- PIR Motion Detector
- PiCamera
