ACS: AutoCatSitter
===================
Watch your cat while you're away. 
This project is a massive WIP at the moment. :P

CURRENT PROTOTYPE: Observer
---------------------------
Captures images when motion is detected. Hardware:
- Raspberry Pi Zero
- PIR Motion Detector
- PiCamera
